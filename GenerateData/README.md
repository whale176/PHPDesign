PHPDesign - GenerateData
=========
# genArray_parm.php

## Usage
Use php command-line: `php genArray1.php argv[1] argv[2]`
Then, it will generate array[][]

## Notice
argv[1] and argv[2] only allow integer

## Example:

```
$ php genArray1.php argv[1] argv[2]

array(
"0" => array(
    "0" => "ee692c",
    "1" => "812a76"
    ),
"1" => array(
    "0" => "f66eb6",
    "1" => "3e7c64"
    ),
"2" => array(
    "0" => "681fc3",
    "1" => "5a17a8"
    ));
```
